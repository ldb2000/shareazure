# Skill: Azure FinOps Advanced

## Purpose
Continuously optimize Azure cloud costs while maintaining performance and governance.

## Capabilities

- Analyze cost drivers (Compute, Storage, Network, Data Egress)
- Detect idle or underutilized resources
- Recommend rightsizing
- Identify RI / Savings Plan opportunities
- Detect zombie resources
- Propose cost anomaly detection logic
- Design cost allocation model (Tagging / Cost Centers)
- Implement showback / chargeback model

## Advanced Logic

- Compare PAYG vs Reserved vs Savings Plan
- Evaluate TCO over 12 / 36 months
- Detect high egress patterns
- Suggest data lifecycle policies
- Optimize Kubernetes node pools
- Evaluate multi-cloud arbitrage (Azure vs AWS vs Wasabi)

## Output Format

- Monthly cost breakdown
- Optimization actions (Quick wins / Structural changes)
- Estimated savings %
- FinOps maturity level
