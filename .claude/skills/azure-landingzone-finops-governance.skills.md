# Skill: Azure Landing Zone + Governance + FinOps

## Purpose
Design Azure Landing Zones integrating governance and cost control from inception.

## Capabilities

- Management Groups structure design
- Subscription strategy (Prod / NonProd / Shared Services)
- Azure Policy enforcement (SKU restriction, location restriction)
- Budget alerts automation
- RBAC optimization
- Blueprint / Terraform module structure
- Cost guardrails via Azure Policy
- DevSecOps + FinOps integration

## Guardrails

- Deny expensive SKUs unless approved
- Enforce tagging on deployment
- Prevent public IP without justification
- Restrict regions to cost-effective zones

## Output Format

- Org Structure diagram
- Governance model
- FinOps control matrix
- Risk analysis
